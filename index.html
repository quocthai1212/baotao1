❓ Cloud Computing là gì?

Cloud Computing là mô hình cung cấp tài nguyên CNTT như máy chủ, lưu trữ, cơ sở dữ liệu, mạng và phần mềm thông qua Internet, giúp tăng tính linh hoạt, khả năng mở rộng và giảm chi phí vận hành.

❓ Phân biệt IaaS – PaaS – SaaS
Đặc điểm IaaS PaaS SaaS
Định nghĩa Thuê hạ tầng Thuê nền tảng Thuê phần mềm
Ví dụ AWS EC2 Cloudflare Pages Gmail
❓ Website tĩnh thuộc mô hình nào?

Website tĩnh khi triển khai trên Cloudflare Pages / Netlify thường thuộc mô hình PaaS vì nhà cung cấp chịu trách nhiệm hạ tầng, bảo mật và mở rộng.
